# Student Course Management Database

This project demonstrates a simple relational database built using SQL.

## Description
The database manages students, courses, and enrollments using relational tables and SQL queries.

## Tables
- Students
- Courses
- Enrollments

## Features
- Create relational tables
- Insert sample data
- Query data using JOIN
- Count students per course

SQL  
Relational Database Design
